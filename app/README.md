# Diagramador Picmimos / Proalbuns — V3.2 Config Base

Esta versão cria uma base central de configuração para o editor, preservando a V5.7 funcional.

## O que mudou

- Novo arquivo: `src/config/productConfigs.js`.
- Modelos de capa cadastrados.
- Formatos e medidas abertas Fuji corrigidas.
- Revestimentos/texturas cadastrados.
- Regra de lombada centralizada: 1 mm por lâmina.
- Painel esquerdo lê as opções da configuração.

## O que não foi mexido

- Fluxo principal do editor 2D.
- Timeline de fotos.
- Crop/enquadramento.
- Botão mover quadro.
- Troca de fotos entre containers.
- Texto editável.
- Layout automático.
- Visual Apple-like.
- Preview 3D ainda não foi refinado.
