# Diagramador Picmimos V3.9

Preview 3D Real MVP em Three.js, mantendo a base configurável para futuro plugin WordPress/WooCommerce.

Ajustes principais:

- Mantém a base da V3.7/V3.8: gabarito de capa, arte inteira para Capa Fotográfica Personalizada e configuração 3D por produto.
- Troca a simulação CSS do preview por uma cena real em React Three Fiber / Three.js.
- O álbum agora é montado por peças físicas: capa, lombada, bloco de lâminas, página/spread, materiais, luz e sombra.
- Capa Fotográfica Personalizada usa arte completa; demais modelos usam textura no verso/lombada e foto na frente.
- Mantém controles: Capa, Aberto, Lombada, Verso, zoom e reset.
- A configuração 3D continua preparada para ser alimentada pelo plugin do laboratório/encadernadora.
